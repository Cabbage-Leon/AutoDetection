# 🎯 智能屏幕监控助手

> 这是一个让你的电脑屏幕变得"聪明"的小工具！它能实时监控你的屏幕，识别各种物体，就像给你的电脑装上了"火眼金睛" 👀

## 🌟 主要功能

### 1. 实时屏幕监控
- 像看直播一样实时查看你的屏幕
- 流畅的画面传输（30FPS，比某些视频网站还流畅！）
- 支持全屏和区域监控

### 2. 智能物体检测
- 使用 YOLO 模型识别屏幕上的各种物体
- 实时标注检测到的物体（就像给它们画了个圈圈 ⭕）
- 显示检测置信度（告诉你它有多确定）

### 3. 自动化操作
- 支持鼠标和键盘的自动化操作
- 可以录制和回放操作序列
- 支持定时任务（让你的电脑自己"干活"）

## 🚀 快速开始

### 环境要求
- Python 3.8+
- CUDA 支持（如果你有 NVIDIA 显卡）
- 一个能正常工作的显示器（废话！）

### 安装步骤

1. 克隆仓库（如果你知道怎么用 Git）：
```bash
git clone https://github.com/yourusername/screen-monitor.git
cd screen-monitor
```

2. 安装依赖（就像给机器人装零件）：
```bash
pip install -r requirements.txt
```

3. 运行程序（见证奇迹的时刻）：
```bash
python app.py
```

4. 打开浏览器访问：`http://localhost:5000`

## 🛠️ 技术栈

- **后端**：Flask（轻量级但功能强大）
- **前端**：HTML + JavaScript（简单但够用）
- **AI 模型**：YOLO（让电脑变得"聪明"）
- **图像处理**：OpenCV（处理图像的瑞士军刀）
